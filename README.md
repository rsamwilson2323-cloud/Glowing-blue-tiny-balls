# ✨ Glowing Blue Tiny Balls

An interactive particle animation created using **HTML5 Canvas** and **JavaScript**. Touch, click, or drag across the screen to generate beautiful glowing blue particles that leave a smooth moonlight-like trail effect.

## 🌟 Features

* ✨ Real-time glowing particle generation
* 🖱️ Mouse interaction support
* 📱 Touchscreen support
* 🌙 Smooth moonlight-inspired visual effects
* 🎨 Dynamic particle colors and glow
* ⚡ Lightweight and dependency-free
* 🖥️ Fullscreen responsive canvas

* Click and drag with your mouse to create glowing particles.
* Touch and move your finger on mobile devices.
* Watch the particles fade smoothly, creating a magical trail effect.
